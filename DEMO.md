## 🎯 **Como Testar**

### **1. Iniciar o Sistema:**
```bash
# Terminal 1 - Backend
cd backend && npm run start:dev

# Terminal 2 - Frontend
cd frontend && npm run dev
```

### **2. Acessar o Painel:**
```
http://localhost:3001
```

### **3. Navegar pelas Seções:**
1. 📊 **Dashboard** - Ver métricas
2. ⚽ **Times** - Gerenciar times
3. 🏆 **Competições** - Gerenciar competições
4. 📅 **Jogos** - Gerenciar partidas
